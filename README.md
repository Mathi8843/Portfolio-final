# Mathi's Portfolio

A minimal, high-performance portfolio website built with Vite and Vanilla JavaScript.

## Features
- ⚡ **Fast**: Built with Vite, zero heavy dependencies.
- 🎨 **Modern Design**: Dark mode, glassmorphism, and smooth animations.
- 📱 **Responsive**: Looks great on mobile and desktop.
- ♿ **Accessible**: Semantic HTML and good contrast.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Dev Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Customization
- **Content**: Edit `index.html`.
- **Styles**: Edit `src/style.css` (check `:root` for variables).
- **Logic**: Edit `src/main.js`.
